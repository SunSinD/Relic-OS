# Notice

Relic OS is a fork-oriented project inspired by TempleOS. This repository starts
with documentation, planning, and project structure so that upstream source can
be studied and imported deliberately.

Before importing upstream source code, record:

- The exact upstream snapshot or archive.
- The original license text.
- The file provenance.
- Any local modifications.
- Any third-party assets or tools needed to build the system.

The existing repository license applies to original Relic OS material unless a
file says otherwise.
