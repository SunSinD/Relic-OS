# Release Process

Relic OS does not have releases yet. This document defines what a future release
should contain.

## Release Contents

- Source archive.
- Build instructions.
- Bootable image if available.
- Checksums.
- Changelog.
- Compatibility notes.
- Known issues.
- Upgrade or migration notes.

## Release Criteria

Before the first release:

- The source provenance is documented.
- The build is reproducible on a clean machine.
- The emulator boot command is documented.
- Known compatibility gaps are listed.
- The license situation is clear.
