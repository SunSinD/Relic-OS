# TempleOS Notes

This page tracks what Relic OS needs to understand about TempleOS before making
large changes.

## Notes To Collect

- Exact upstream version or archive.
- License text and source origin.
- Boot path.
- Kernel layout.
- Filesystem behavior.
- HolyC language and compiler behavior.
- Shell and editor workflow.
- Graphics and input model.
- Programs or demos that should remain compatible.

## Compatibility Labels

- `keep`: preserve this behavior.
- `change`: modernize or replace this behavior.
- `study`: not understood yet.
- `drop`: intentionally not supported.
