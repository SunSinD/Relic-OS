# Glossary

## ADR

Architecture Decision Record. A short document that records a decision, its
context, and its consequences.

## ABI

Application Binary Interface. The binary-level contract between compiled code
and the system it runs on.

## Bootloader

Code that prepares the machine and transfers control to the operating system.

## Kernel

The core part of the operating system responsible for hardware control and
system services.

## Provenance

The recorded origin of code, assets, tools, or data.

## Userland

Programs, libraries, shells, and tools that are not part of the kernel.
