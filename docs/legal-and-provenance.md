# Legal And Provenance

Operating system repositories often collect code, firmware assumptions, tools,
fonts, images, and test media. Track where everything came from.

## Required For Imported Code

- Source name.
- Source URL or archive name.
- Snapshot date.
- Commit hash or checksum when available.
- License text.
- Local modifications.
- Person who imported it.

## Required For Assets

- Asset name.
- Author or source.
- License.
- Whether modification is allowed.
- Whether redistribution is allowed.

## Upstream TempleOS Checklist

- [ ] Pick exact upstream snapshot.
- [ ] Store license text.
- [ ] Record archive checksum.
- [ ] Build source inventory.
- [ ] Identify generated files.
- [ ] Identify third-party files.
- [ ] Decide import strategy.
